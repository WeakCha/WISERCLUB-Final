Preface

This website consists of several parts:

Project Description (Part 1 - Part 3)
This is a comprehensive data science project for the Final of WISERCLUB 2019-2020. You have got two csv files, named data.csv and holiday.csv. Try to use Python or SQL to complete the tasks with the aid of these two datasets. Please keep in mind that Data Science or Business Analytics is to use data and models to find hidden information.

This project consists of three parts: Explorative Data Analysis, Data Preprocessing and Model Training and Prediction. The final decision about whether one will be admitted in WISERCLUB or not DOES NOT solely depend on the project.

Hints
There will be some hints for the project. Some problems require close analysis and observations, therefore some related hints are not shown at the beginning. The days for them to be shown on the website are marked with bold words, but please try to find answers by yourself before that time.

Numerical Experiments (Part 4)
Some numerical computing tricks in Python are listed in this part as problems. Although Part 4 is not considered as a part of Final Project, there are some interesting findings if you have time solve those problems.

Now, you could begin your exploration in the ocean of data.
